# Pokemon Explorer

Prosta aplikacja webowa do przeglądania i wyszukiwania pokemonów, zbudowana na bazie publicznego API [PokeAPI](https://pokeapi.co).

## Funkcje

- Przeglądanie listy pokemonów z paginacją
- Wyszukiwanie po nazwie (z debouncingiem)
- Filtrowanie po typie
- Szczegółowy widok pokemona (statystyki, typy, zdolności)
- Dodawanie pokemonów do ulubionych (LocalStorage)

## Technologie

- [Next.js 15](https://nextjs.org) (App Router)
- [React Query](https://tanstack.com/query) - pobieranie i cachowanie danych
- [Tailwind CSS](https://tailwindcss.com) - stylowanie
- [shadcn/ui](https://ui.shadcn.com) - komponenty UI

## Uruchomienie

```bash
npm install
npm run dev
```

Aplikacja dostępna pod [http://localhost:3000](http://localhost:3000).

## Możliwy rozwój

- Animacje przejść między kartami i stronami (np. Framer Motion)
- Animowane pojawianie się kart pokemonów

# Pokemon Explorer _(English)_

A simple web app for browsing and searching Pokémon, built on top of the public [PokeAPI](https://pokeapi.co).

## Features

- Browse Pokémon list with pagination
- Search by name (with debouncing)
- Filter by type
- Detailed Pokémon view (stats, types, abilities)
- Add Pokémon to favourites (LocalStorage)

## Tech Stack

- [Next.js 15](https://nextjs.org) (App Router)
- [React Query](https://tanstack.com/query) - data fetching and caching
- [Tailwind CSS](https://tailwindcss.com) - styling
- [shadcn/ui](https://ui.shadcn.com) - UI components

## Getting Started

```bash
npm install
npm run dev
```

App available at [http://localhost:3000](http://localhost:3000).

## Possible Future Development

- Page and tab transition animations (e.g. Framer Motion)
- Animated Pokémon card entrance effects
